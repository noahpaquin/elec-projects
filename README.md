### Screen Vision Automation Tool
**Language:** Python  
**Libraries:** OpenCV, mss, pyautogui, pynput, NumPy

A real-time screen capture and computer vision tool that uses template matching 
to detect visual events and trigger automated responses. Built around a 
finite state machine (SEARCHING → CLICKING) with configurable timing, 
multi-threaded keyboard input handling, and ROI-based image processing for 
performance optimization.

**Key concepts:** template matching, state machines, real-time image processing, 
multi-threading, event-driven programming
